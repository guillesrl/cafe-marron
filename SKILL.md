# Frontend Design Guidelines

This project follows the frontend-design skill guidelines for creating distinctive, production-grade interfaces.

## Design Direction

- **Purpose**: Premium coffee shop landing page showcasing artisanal coffee experience
- **Tone**: Art deco / geometric luxury - sophisticated, warm, memorable
- **Differentiation**: Rich earthy palette, geometric accents, grain textures, distinctive typography

## Aesthetics Guidelines

### Typography
- Cormorant Garamond: elegant serif display font with italic variants
- Outfit: modern geometric sans-serif for body text
- Large display sizes, generous letter spacing

### Color & Theme
- Deep espresso background (#1a1612)
- Warm cream text (#f5e6d3)
- Copper/terracotta accents (#c47f5e)
- CSS variables for consistency

### Motion
- Elaborate GSAP timeline on page load
- Staggered reveals with animation-delay
- Scroll-triggered animations per section
- Hover transforms on cards and images
- Smooth scroll to anchors

### Spatial Composition
- Two-column asymmetric layouts
- Grid systems for menu and timeline
- Generous negative space
- Layered elements with depth

### Visual Details
- SVG grain/noise texture overlay
- Geometric patterns and borders
- Decorative accent elements
- High-quality Unsplash imagery
- Subtle box shadows and borders

## Avoid
- Generic "AI slop" aesthetics
- Inter, Roboto, Arial, system fonts
- Purple gradients, generic dark themes
- Cookie-cutter layouts
